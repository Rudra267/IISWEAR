Programme Overview

The Certificate in Corporate Performance Management is a comprehensive online programme tailored for finance professionals, analysts, and aspiring CFOs. The curriculum is designed to impart deep knowledge of relational databases, data warehousing, data modeling, ETL systems, and financial analytics—all necessary to thrive in a performance-driven financial leadership role. Through an immersive, case-based structure, you will build end-to-end data systems and leverage predictive analytics to drive financial decisions.

Programme Modules

Module 0 – Relational Database Management Systems (RDBMS) Concepts

  • Advanced Introduction to DBMS and EDBMS
  • DBMS vs RDBMS vs NoSQL
  • Codd’s 12 Rules for RDBMS
  • Entity Relationships in RDBMS
  • Keys and Normalization (1NF to 4NF)
  • Views and Materialized Views
  • Advanced RDBMS for Cloud Analytics
  • Project: OLTP System Design

Module 1 – Data Warehousing Concepts for Financial Analytics

  • Introduction to DWBI
  • Basics of Statistics for Analytics
  • Business Intelligence Frameworks
  • Application of BI in Financial Domains
  • Project: Client BI Architecture Design

Module 2 – Data Modeling for CFO Office

  • Logical and Physical Data Modeling
  • Star & Snowflake Schema Development
  • Domain-Specific Data Warehouse Schema
  • Project: Data Warehouse Design for Analytics

Module 3 – Data Cleansing and ETL for Financial Planning

  • Domain Data Ecosystem Overview
  • ETL (Extract, Transform, Load) Frameworks
  • Big Data for Financial AI
  • Introduction to Data Engineering
  • AWS and Azure Ecosystem Essentials
  • Project: ETL Implementation for Financial Quality

Module 4 – Planning, Budgeting & Forecasting

  • Integration of Concepts into Financial Planning
  • Introduction to CPM Implementation
  • Capstone Project: Consolidated Business Planning Model

Module 5 – Mining, Visualization, and CPM Roadmap

  • Financial Reporting Tools & Dashboards
  • Project: Data Mining for Performance Reporting

Module 6 – Boot Start Your Career

  • Work on International Projects
  • Internship: Real-time experience before client office onboarding
