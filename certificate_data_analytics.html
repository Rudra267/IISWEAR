Programme Overview

The Certificate in Corporate Performance Management is a comprehensive online programme tailored for finance professionals, analysts, and aspiring CFOs. The curriculum is designed to impart deep knowledge of relational databases, data warehousing, data modeling, ETL systems, and financial analytics—all necessary to thrive in a performance-driven financial leadership role. Through an immersive, case-based structure, you will build end-to-end data systems and leverage predictive analytics to drive financial decisions.

Programme Modules

Module 0 – Application of RDBMS Concepts

  • Advanced Introduction to DBMS and EDBMS
  • DBMS vs RDBMS vs NoSQL Databases
  • 12 Codd’s Rules for RDBMS
  • Entity Relationships in RDBMS
  • Keys in RDBMS
  • Normalization (1NF to 4NF)
  • Views and Materialized Views
  • Cloud-Based RDBMS for Analytics
  • Project: OLTP System Design

Module 1 – The Data Warehousing and BI Ecosystem

  • Introduction to DWBI
  • Basics of Statistics for Analytics
  • Application of Business Intelligence
  • Business Intelligence Solution Frameworks
  • Project: BI and Analytics Solution Architecture

Module 2 – Aspects of Data Modelling

  • Introduction to Data Modelling
  • Logical Modelling for DW & BI
  • Physical Modelling Techniques
  • Star & Snowflake Schemas
  • Domain Data Warehouse Schema for Advanced Analytics
  • Project: Schema Design for Analytics using Star & Snowflake

Module 3 – Data Engineering Ecosystem

  • Domain-specific Data Ecosystems
  • ETL Concepts and Frameworks
  • Introduction to Big Data and AI
  • Data Engineering Essentials
  • AWS & Azure Data Services
  • Project: ETL Pipeline for Analytics

Module 4 – Advanced Data Analytics Landscape

  • Capstone Project covering Modules 0–3
  • BI Project Management Basics

Module 5 – Mining & Visualization Techniques

  • Data Visualization Techniques and Tools
  • Project: Stakeholder Reporting Dashboard

Module 6 – Boot Start Your Career

  • Work on International Projects with Global Clients (Remote/Hybrid Model)
